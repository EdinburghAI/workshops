# Neural Networks

This workshop moves from the intuition of a single neuron to training a small neural network in PyTorch. It is the bridge between classical ML from Workshop 1 and the deep-learning workshops that follow.

## What Students Build

- A single linear neuron using `torch.nn.Linear`.
- A small multilayer network using `nn.Sequential`.
- A hand-written training loop with loss, optimizer steps, and backpropagation.
- A first MNIST classifier using PyTorch datasets and dataloaders.

## Run It

This notebook can run on Kaggle or locally.

For Kaggle:

1. Import the notebook from GitHub.
2. Keep internet enabled so the MNIST dataset can download.
3. Run the notebook from top to bottom.

For local use, install PyTorch and torchvision in your environment before opening the notebook.
